Cost Estimator

  2. Portfolio Details

Project information

  • Category: Web Application
  • Tech Stack: Node.js, Handlebars, JQuery, JavaScript
  • Database: MongoDB

A cost estimator had been developed for runtime cost calculation during placing order of printing products like poster, banner, wedding cards.

Printing of different products goes through several steps wherein each step a good number of raw materials are used. Again, there is a lot of variation for a single raw material which ultimately affects the final cost. This relation between raw materials and the final product is dynamic and unique for each and every product. That's why MongoDB has been used as NoSQL database to accommodate this relation for any printing company regardless of products. Admin can configure the relation which will be ultimately followed during product order to suggest an estimated price to the consumer.

The existing features in this application:
  • New Product: Admin can create new product which they will offer to the customers.
  • New Raw Material: New raw material can be included in the system by the admin.
  • Production Procedure: Admin can entry full production procedure and edit any time.
  • Cost Calculator: Consumer can view the price of his expected products trying differnt combinations. Any change of input will update the current price.
  • Generic Solution: This solution is not aimed for a single company. Any printing comapany can use this application by configuring the systems by themselves. This flexibility was its main objective.

Project information

  • Category: Game
  • Tech Stack: IGraphics

KenKen was my first project as undergrad student.

KenKen is a number puzzle game following some rules to solve the puzzle. The complexity of the game depends on the size of the grid. The grid may contain any number of cells like (2*2, 5*5, 9*9). No number will be repeated in any row or column. For 2*2 grid, the cells need to be populated with 1,2. For 3*3, it will be 1,2,3, and 1,2,....n will be used for n*n grid.

The features I provided in this game.
  • Level: 3 levels were added.
  • Level1: 3*3 puzzle.
  • Level2: 4*4 puzzle.
  • Level3: 5*5 puzzle.
  • Timer: A timer was set for a particular level. Gamer had to solve the puzzle within this time.
